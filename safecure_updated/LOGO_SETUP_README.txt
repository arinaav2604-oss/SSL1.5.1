  SAFECURE WEBSITE — LOGO SETUP INSTRUCTIONS
═══════════════════════════════════════════════════════════════

Two logo image files need to be placed at:

  /assets/logo-header.png   ← Header (navbar) logo
  /assets/logo-footer.png   ← Footer logo

WHERE TO PUT THEM
─────────────────
Create an "assets" folder in the root of this website folder
(same level as index.html) and place your PNG files there:

  safecure_updated/
  ├── assets/
  │   ├── logo-header.png   ← header logo
  │   └── logo-footer.png   ← footer logo
  ├── index.html
  ├── team.html
  ├── ...

LOGO SPECIFICATIONS
───────────────────
Header logo (logo-header.png):
  - Height: 40px (width auto)
  - Background: Transparent PNG recommended
  - Colour: White / light version (placed on dark navy bg)

Footer logo (logo-footer.png):
  - Height: 44px (width auto)
  - Background: Transparent PNG recommended
  - Colour: White / light version (placed on dark slate bg)

FALLBACK BEHAVIOUR
──────────────────
If the PNG files are not found, the site will automatically
show a text fallback "SAFECURE / Services Limited" in the
correct styling — so the site will still look good while
you're setting up the logos.
